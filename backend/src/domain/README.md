Estructura de Dominio (Domain)

Este directorio contiene entidades puras y contratos (interfaces) por módulo de dominio:
- authors (autores, países)
- books (libros, libros_x_autores, libros_x_generos, géneros)
- users (usuarios, estados)
- voices (voces, tonos, idiomas, géneros de voz, voces_x_tonos)
- progress (libros_x_usuarios, playbackrate)
- publications (publicados)

Notas:
- Nombres en español alineados al esquema de base de datos.
- Archivos sin lógica; sólo placeholders para estructuración.
